## Instrucciones para compilar y ejecutar los ejercicios:

## Compilar:

1. Abrí la terminal.
2. Ejecutá le comando "make": ```make```

## Ejecutar Ejercicio 1:

1. Ejecutá el comando: ```./Ejercicio1```

## Ejecutar Ejercicio 2:
